## TL;DR (AI Summary)
Using nonparametric regression techniques (Neural Networks and Thin Plate Splines) it is possible to consistently estimate the dominant Lyapunov exponent (LE), and these methods have been studied using simulated data and applied to a biological time series: marten fur returns for the Hudson Bay Company.
